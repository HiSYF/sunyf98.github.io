#### 创建

#### 清空缓存

```tex
hexo clean
```

#### 生成模板

``` tex
hexo g
```

#### 上传到服务器

```tex
hexo d
```

#### 在本地运行

``` tex
hexo s
```

#### 新建文章

```tex
hexo n "文章名"
```

#### 新建页面

```tex
hexo n page "页面名"
```

#### 结合命令

```tex
# 生成并上传
hexo d -g 
```
